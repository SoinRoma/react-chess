![React](https://img.shields.io/badge/-React-black?style=flat-square&logo=react)

### Запуск проекта для Windows

1-Запустить проект в WebStorm

2 - Установить все зависимости(IDE сам придложит).
```
npm install
```
3 - Запуск проекта
```
npm start
```
